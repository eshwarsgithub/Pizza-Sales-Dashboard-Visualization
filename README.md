# Pizza-Sales-Dashboard-Visualization
This project involves creating an interactive Power BI dashboard to analyze a pizza sales dataset. The goal is to provide key insights into sales trends, customer behavior, and product performance.

Key Skills and Technologies Used:

SQL: Utilized for data cleaning, aggregation, and querying the raw sales data to create a structured dataset.

Power BI: Employed for data modeling, creating DAX measures, and designing a dynamic dashboard with various visualizations to present the findings effectively.

The dashboard provides a comprehensive view of the business, enabling data-driven decisions and highlighting best-selling items, revenue trends, and other critical business metrics
